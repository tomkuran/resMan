Why was this built?

This starter kit is greatly inspired and built upon the original starter app provided by angular/material-start located at angular/material-start.

The main purpose of building this starter kit was to provide a base for building and producing quick, material design interfaces with AngularJS. While there are various starter kits available, this one tries to mimick the original material design specification by Google closely, and more and more features will be added to it in terms of transitions, animations, components and more.

Angular Material

Built using Angular Material, there are over 32 material design components that you can leverage right away. The Angular Material implementation of Google's Material Design provides a set of reusable, well-tested, and accessible UI components. To know more, visit the documentation.
